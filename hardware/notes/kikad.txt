IMPLEMENTATION NOTES (v1.0)
============================

Board Dimensions: 140 mm × 70 mm (widened from 120 mm to accommodate 21 resistors in a row)

Component Placement:
- J1 (21-position terminal block): Centered at (60, 10) — harness pads aligned
- RDRV0–RDRV20 (resistors): Row at y=25, from x=15 to x=135, spaced 6mm apart
- RREF (1kΩ 1% precision): Placed at (70, 50) for resistance measurement circuit
- U1 (SparkFun Artemis ATP): Centered at (60, 50) — schematic symbol references
- Mounting holes (H1–H4): Corners at (5,5), (135,5), (5,65), (135,65)
- TP_RES_NODE (test pad): At (80, 50) for probing or calibration

Netlist:
- CH0–CH20: Test node nets (J1 pads 1–21)
- U1_PIN2–U1_PIN22: Artemis GPIO pins 2–22 mapped to channels 0–20
- RES_DRIVE: U1.PIN23, drives RREF for resistance measurement
- RES_NODE: Voltage divider node (connected to RDRV0 input via net U1_PIN2 for CH0 measurement)
- GND: Common return (harness and board)
- +3.3V: Power rail (optional future for pull-ups or termination)

Resistance Measurement Circuit:
- U1.PIN23 (RES_DRIVE) → RREF (1kΩ) → RES_NODE (voltage divider node)
- RES_NODE connects to:
  - RDRV0 input (Artemis side), enabling resistance measurement across CH0
  - Test pad TP_RES_NODE for direct probing or oscilloscope calibration
- Virtual divider forms with unknown resistance from RES_NODE to GND via harness under test

Firmware Implementation:
- PIN23 is configured as digital output (RES_DRIVE)
- A0 is configured as analog input (RES_NODE voltage sense)
- measure_resistance() function drives PIN23 HIGH and reads voltage at RES_NODE
- Formula: Rx = (Rref × Vread) / (Vcc - Vread)
- Supported for CH0 in v1.0; future versions can add multiplexer for channels 1–20

Notes for Manufacturing:
- All resistors are 0805 SMD or 0603 (adjust footprints as needed)
- RREF should be 1% metal film for accuracy (e.g., Vishay MRS025 1% series)
- PCB is 2-layer (F.Cu, B.Cu); no routing added yet (user can auto-route in KiCad)
- Edge connector J1 uses standard 5.08mm pitch (Phoenix or equivalent)
- Artemis footprint uses SparkFun-Boards library (downloaded separately)
